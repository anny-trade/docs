# Compliance & Disclaimers

## Financial Disclaimer

All data returned by Anny's MCP tools is informational. It is not financial advice.

Every analytical tool response includes this disclaimer:

> *This is informational data from Anny's indicators, not financial advice. Past performance does not indicate future results. Always do your own research before making investment decisions.*

## What Anny Does and Does Not Do

| Anny does | Anny does not |
|-----------|---------------|
| Provide indicator readings | Give buy/sell recommendations |
| Show your portfolio status | Execute trades or move funds |
| Run historical scenario analysis | Predict future prices |
| Answer questions about market data | Manage your exchange accounts |

## Data & Privacy

- Anny accesses your portfolio data only when you explicitly call a portfolio tool
- Exchange API keys and credentials stay on Anny's infrastructure — they are never sent through MCP responses
- MCP tool responses contain only the formatted results, not raw API data
- No data from your Claude conversations is stored by Anny
- See our [Privacy Policy](https://anny.trade/privacy) for full details

## Read-Only Access

All MCP tools are read-only. The connector cannot:
- Place orders on any exchange
- Modify your Anny Trade account settings
- Transfer any assets
- Access your exchange API keys

The only write operation is the feedback tool, which sends your feedback text to our development team.
